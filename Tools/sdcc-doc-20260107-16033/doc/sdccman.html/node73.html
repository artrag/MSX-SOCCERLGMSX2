3.5.4 Rabbit intrinsic named address spaces


3.5.4.1 __sfr

The Rabbit ports support a 64 KB __sfr address space corresponding to the ”internal i/o”space of the Rabbits.


3.5.4.2 __far

The Rabbit ports supports a 1 MB __far address space that contains the generic address space. Due to limitations of the Rabbit ldp instructions, variables declared as __far can be neither volatile nor atomic. Note: This support is still very incomplete and should be considered experimental.
