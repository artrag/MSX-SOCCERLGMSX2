Scrivi uno script python che carica una immagine PNG 256xL, con palette a 16 colori, 
Assumiamo L pari a 504

Sia K l'altezza di una striscia con dimensione 256xK, assumi K=16 come default, ma consenti di modificare il parametro linea di comandi

Scompone l'immagine in strisce 256xK partendo da ciascuna delle 504-K+1 ordinate (non prendere strisce che escono fuori dall'immagine in ingresso)

Scrivi la striscia con ordinata 0 (righe da 0 a K-1) in un file immagine di output e registra che la prima striscia dell'input si codifica con la ordinata 0

Scorri verticalmente la striscia con ordinata 1 (righe da 1 a K) nel file immagine di output: 
- se coincide con la striscia tra 0 e (K-1) registra che striscia con ordinata 1 si codifica con l'ordinata 0
- se coincide con la striscia tra 1 e (K-1) e l'ultima riga cade nella regione vuota dell'immagine output, aggiungi l'ultima riga di seguito all'ultima riga  dell'immagine di output e registra che la striscia 1 si codifica con la ordinata 1

Scorri verticalmente la striscia con ordinata 2 nel file immagine di output:  
- se coincide con la striscia tra 0 e (K-1) registra che striscia con ordinata 2 si codifica con la ordinata 0
- se coincide con la striscia tra 1 e K registra che striscia con ordinata 2 si codifica con la ordinata 1
- se coincide con la striscia tra 2 e K e l'ultima riga cade nella regione vuota dell'immagine output, aggiungi l'ultima riga di seguito all'ultima riga del file di output e registra che la striscia 1 si codifica con la ordinata 2

In generale:
- Scorri verticalmente la striscia con ordinata n (righe da n a n+K-1) nel file immagine di output:  
- se coincide con la striscia tra y e y+K-1 registra che striscia con ordinata n si codifica con la ordinata y
- se coincide con le utime K-1 righe nel file immagine di output (tra k e k+K-2 dove k+K-2 è l'utima riga dell'immagine ) aggiungi l'ultima riga di seguito all'ultima riga del file di output e registra che la striscia con riga n si codifica con ordinata k

Se non c’è match completo e non c’è match sulle ultime K-1 righe, aggiungi l'intera striscia in coda all'immagine di output, registra che la striscia con riga n si codifica con ordinata dell'utima striscia aggiunta e continua l'algoritmo

Quando la striscia da n a n+K-1 supera la fine dell'immagine (ossia n+K-1>L-1) riduci la dimensione di K in modo da non considerare linee fuori dall'immagine.
(assumi K=L-n).

Salva come file ASM il file che codifica in quale striscia dell'immagine di output si codifica ciascuna striscia del file di input.

Salva come array C il file che codifica in quale striscia dell'immagine di output si codifica ciascuna striscia del file di input.
Usa come nome dell'array FieldMap[] e come tipo u8 (che equivale a unsigned char nel mio progetto)
Denomina questo file FieldMap.h

Se l'immagine in output ha fino a 256 linee usa .DB invece di .DW

Salva l'immagine di output come PNG con la medeisima palette dell'ingresso

Salva la conversione in binario dell'immagine di output ottenuta impacchettando due pixel per byte. 
Usa l'estensione BIN.

Crea una cartella (default "outdir")  in cui inserisci i file di output (PNG, BIN e BIN)
